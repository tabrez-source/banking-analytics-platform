\## Dataset Information



This project uses the \*\*Synthetic Financial Transactions for Fraud Detection\*\* dataset from Kaggle.



Due to GitHub's file size limits, raw data files are not stored in this repository.



\### How to Run the Project

1\. Download the dataset from Kaggle.

2\. Place the CSV file locally at:

&nbsp;  C:\\SQLData\\transactions.csv

3\. Execute SQL scripts in the following order:

&nbsp;  - staging/

&nbsp;  - oltp/

&nbsp;  - dw/
